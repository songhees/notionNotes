Tomcat

64-bit Windows zip (pgp, sha512) 설치

환경설정

window → preference → server → runtime enviroment → add → tomcat10

  • tomcat설치 경로를 이클립스에 알려 이클립스에서 이를 껐다 켰다 할 수 있게 하는 것
    1. Open perspective에서 java EE → servers 에 Tomcat 10을 입력해 웹 서버를 구성
    1. admin port에 Port Number값을 입력
    1. 저장 후 start 실행
    1. 새로운 프로젝트 생성 → dynamic web project 실행
    1. target runtime이 tomcat이어야 한다.
    1. 생성한 프로젝트에서 src/main/java 폴더가 소스 적는 곳
    1. webapp에 → jsp → 파일 형성

  • 배포할 프로젝트를 웹서버에 등록하기
    1. server에서 오른쪽 버튼 누른뒤 add and remove실행
    1. 만든 프로젝트를 add하여 finish한다.
    1. server의 Tomcat안에 해당 프로젝트가 있다.
    1. 브라우저에

      http://localhost:8080/sample/home.jsp ← 파일 경로를 입력하면 프로젝트로 만든 웹 애플리케이션이 나온다.

      tomcat double click → module → path에 나오는 식별자 ex) /sample

      웹서버에 배포된 웹 애플리케이션(프로젝트)의 식별자

      http 프로토콜이 사용하는 웹서버의 기본 port번호 80

프로젝트 안의 폴더

src/main/java 자바 패키지, 클래스, 인터페이스 정의하는 곳

src/main/resources 자바 프로그램 실행에 필요한 환경 설정 파일을 정의하는 곳

src/test/java 테스트 케이스를 정의하는 곳(src/main/java에서 정의한 프로그램을 테스트하는 프로그램 정의)

src/test/resources 테스트 케이스 실행에 필요한 환경 설정 파일을 정의하는 곳

src/main/webapp 폴더, 이미지, CSS파일, JavaScript 파일, JSP파일을 정의하는 곳

  • 외부에 접근이 차단된 폴더(숨겨진 폴더)

    src/main/webapp/META-INF 웹 애플리케이션 실행에 필요한 부가 적인 설정 파일을 정의하는 곳

    src/main/webapp/WEB-INF 웹 애플리케이션 실행에 필요한 설정 파일을 정의하는 곳

    src/main/webapp/WEB-INF/lib 웹 애플리케이션 실행에 필요한 jar파일이 위치하는 곳 // 오라클 jdbc파일을 넣는 곳
